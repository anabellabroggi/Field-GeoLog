# Field-GeoLog
Proyecto de analisis de lote, donde incluimos imágenes NDVI, procesamiento de altimetría mediante GEE, incorporación y geoprocesamiento de cartas de suelos, entre otras herramientas para el análisis de lotes mediante el uso de imagenes satelitales. 
# 1. NearestRoad 
Este código calcula el camino más cercano hasta la ruta principal. Toma una coordenada geográfica que se ingresa de forma manual
